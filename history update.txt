[***] GUDANG.ID [***]
aplikasi manajemen data gudang dan data penjualan

[v0.0] analisa dan perancangan fitur-fitur awal dan siapa yang akan menggunakannya.
[*] level user yang akan mengakses aplikasi
    1. Manager
    2. Super Admin
    3. Admin

[*] Fitur-fitur (halaman apa saja yang akan ada didalam sistem)
    1. Monitoring
    2. Data Item
        - spesifikasi tabel untuk list item
        - Kode barang
        - Nama barang
        - Kategori
        - Harga
    3. Data Pelanggan
        - Spesifikasi tabel
        - Kode Pelanggan
        - Nama Pelanggan
        - Alamat
        - Telepon
    4. Data Transaksi
        - Spesifikasi tabel
        - Kode Barang
        - Nama Barang
        - Kategori
        - Harga barang satuan
        - Quantitas
        - Diskon
        - Total
    5. Data Admin (Admin level pengelola gudang)
    6. Report
